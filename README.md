# Rapport au format Latex pour Centrale Lille
Ce projet a pour but de mettre a disposition un template pour latex pour des rapports à l'Ecole Centrale de Lille.

## Origine
Ce projet est une adaptation du projet suivant : https://github.com/Template-Latex/Template-Informe.git réalisé par Pablo Pizzaro R. à l'Université du Chili.
